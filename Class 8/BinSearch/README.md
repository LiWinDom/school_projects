# Бинарный поиск

Да, тут действительно бинпоиск, просто в неочевидной обёртке.
Программа читает число `n` и выводит `⌊sqrt(n)⌋` — целую часть квадратного корня

## Что делает код

- Использует бинарный поиск по диапазону от `1` до `n`
- Сравнивает через `mid <= n / mid`, чтобы не ловить переполнение на `mid * mid`
- Обрабатывает крайние случаи `0` и `1` отдельно
